Spyder installation

  1. Clamp spyder lockring tool into a bench vise as shown.
  2. Insert crankarm with lockring into tool as shown, ensuring the tool tangs are engaged with the lockring.
  3. While holding crankarm onto tool with one hand, turn crankarm with other hand to loosen or tighten lockring.

Notes:

Lockring has normal right hand threads, turn clockwise to tighten, counterclockwise to loosen.

Locktite can be used on the spyder interface and lockring threads.

Locktite must be used for track bikes and tandem timing side spyders.
